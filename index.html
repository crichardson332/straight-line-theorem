The Straight Line Theorem

Chris Richardson

April 15, 2020

PROBLEM STATEMENT

  • The shortest path \(f(x)\) passing between two points \(x_1\) and \(x_2\) is a straight line.

  • To prove the theorem, we will use a field of mathematical analysis called calculus of variations developed by Bernoulli, Euler, and Lagrange in the 18th century.
  • This field is concerned with finding extrema (maxima and minima) of functionals, which can be thought of as functions of functions.

  • Given a function \(f(x)\), the integral \(\int f(x) dx\) is a functional of \(f(x)\).

Background: Calculus of Variations

  • Consider the functional: \[ J(y) = \int_{x_1}^{x_2} L(x,y(x),y'(x))dx \]
  • The goal is to minimize \(J(y)\). Let \(f = \underset{y}{\mathrm{argmin\,}} J(y)\). Then we can say that for any arbitrary smooth variation \(\eta(x)\) vanishing at the endpoints, \[ J(f) \leq J(f+\epsilon \eta) \] for some \(\epsilon > 0\). Now we define a function of \(\epsilon\) \[ \Phi(\epsilon) \overset{\Delta}{=} J(f+\epsilon \eta) \]
  • This function is minimized at \(\epsilon=0\). Because it's a scalar function we can set the derivative to 0:
  • \[ \frac{d \Phi(0)}{d\epsilon}=0 \]

Background: Calculus of Variations

  • Now we look at the total derivative\[\frac{d \Phi}{d\epsilon} = \int_{x_1}^{x_2} \frac{dL}{d\epsilon} dx = 0\]

  • Taking the total derivative of \(L(x,y,y')\), noting that \(y=f+\epsilon \eta\) and \(y'=f'+\epsilon \eta'\) we get

  • \[\frac{d L}{d\epsilon} = \frac{\partial L}{\partial y}\frac{dy}{d\epsilon} + \frac{\partial L}{\partial y'}\frac{dy'}{d\epsilon}\]
  • And because \(\frac{dy}{d\epsilon}=\eta\) and \(\frac{dy'}{d\epsilon}=\eta'\), we get \[\frac{d L}{d\epsilon} = \frac{\partial L}{\partial y}\eta + \frac{\partial L}{\partial y'}\eta'\]\[\int_{x_1}^{x_2} \frac{dL}{d\epsilon} dx = \int_{x_1}^{x_2} \left(\frac{\partial L}{\partial f}\eta + \frac{\partial L}{\partial f'}\eta'\right) dx = 0\]

Background: Calculus of Variations

  • Using integration by parts we get \[\int_{x_1}^{x_2} \left(\frac{\partial L}{\partial f}\eta + \frac{\partial L}{\partial f'}\eta'\right) dx = \int_{x_1}^{x_2} \frac{\partial L}{\partial f}\eta dx + \frac{\partial L}{\partial f'}\eta \biggr\rvert_{x_1}^{x_2} - \int_{x_1}^{x_2} \eta \frac{d}{dx} \frac{\partial L}{\partial f'} dx\]
  • Because \(\eta\) vanishes at the endpoints, we end up with \[\int_{x_1}^{x_2} \eta \left(\frac{\partial L}{\partial f} - \frac{d}{dx} \frac{\partial L}{\partial f'}\right) dx = 0\]
  • According to the fundamental lemma of the calculus of variations, the integrand here must be identically zero. Thus \[\frac{\partial L}{\partial f} - \frac{d}{dx} \frac{\partial L}{\partial f'} = 0\]
  • This is the Euler-Lagrange Equation.

Original Problem: Path Length

  • Revisiting our original problem, we want to find a function \(y(x)\) that minimizes the path length between two points. In 2-D the path length is given by the functional \[\int_{x_1}^{x_2} \sqrt{1+\left(\frac{dy}{dx}\right)^2} dx\]
  • The Lagrangian here does not explicitly depend on \(x\) or \(y\). Thus the Euler-Lagrange equation reduces to \[ \frac{d}{dx} \frac{\partial L}{\partial f'} = 0\]
  • We can integrate both sides and get \( \frac{\partial L}{\partial f'} = c\). Thus \( \frac{\partial L}{\partial f'} = \frac{y'}{\sqrt{1+y'^2}} = c\). Solving for \(y'\) we get \( y' = \sqrt{\frac{c^2}{1-c^2}} = k\). Because the slope of the line is constant, it is a straight line.
